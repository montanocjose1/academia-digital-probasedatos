🔗 Relaciones entre Tablas

categorias → cursos

Una categoría puede tener muchos cursos. Un curso pertenece a una categoría. Relación 1:N.

usuarios → cursos (instructor)

Un usuario (instructor) puede crear muchos cursos. Un curso tiene un instructor. Relación 1:N.

cursos → capitulos

Un curso tiene muchos capítulos. Un capítulo pertenece a un curso. Relación 1:N con borrado en cascada.

capitulos → lecciones

Un capítulo tiene muchas lecciones. Una lección pertenece a un capítulo. Relación 1:N con borrado en cascada.

lecciones → recursos_multimedia

Una lección puede tener muchos recursos multimedia. Cada recurso pertenece a una lección. Relación 1:N.

lecciones → evaluaciones

Una lección puede tener muchas preguntas de evaluación. Cada evaluación pertenece a una lección. Relación 1:N.

usuarios → progreso_estudiante

Un usuario tiene muchos registros de progreso. Cada progreso pertenece a un usuario. Relación 1:N.

lecciones → progreso_estudiante

Una lección puede tener muchos registros de progreso (de diferentes estudiantes). Relación 1:N.

usuarios → inscripciones

Un usuario puede tener muchas inscripciones a cursos. Relación 1:N.

cursos → inscripciones

Un curso puede tener muchas inscripciones de estudiantes. Relación 1:N.

usuarios → repositorios_github

Un usuario puede tener muchos repositorios GitHub asociados. Relación 1:N.

cursos → repositorios_github

Un curso puede tener muchos repositorios asociados. Relación 1:N.

📖 Descripción de Tablas

TablaDescripción
categoriasClasificación temática de los cursos (IA, Programación, Datos, etc.)
usuariosInformación de todos los usuarios del sistema: estudiantes, instructores y administradores
cursosCursos ofertados en la plataforma con metadatos (precio, nivel, rating)
capitulosMódulos o unidades que componen cada curso
leccionesUnidades de aprendizaje individuales dentro de cada capítulo
recursos_multimediaMaterial complementario: imágenes, videos, diagramas, animaciones
inscripcionesRegistro de estudiantes inscritos en cursos con su progreso
progreso_estudianteSeguimiento detallado del avance del estudiante en cada lección
evaluacionesPreguntas de opción múltiple para evaluar el aprendizaje
respuestas_estudianteRespuestas enviadas por estudiantes a las evaluaciones
repositorios_githubEnlaces a repositorios GitHub asociados a cursos y proyectos
foro_preguntasPreguntas del foro de discusión de cada curso
foro_respuestasRespuestas a las preguntas del foro
